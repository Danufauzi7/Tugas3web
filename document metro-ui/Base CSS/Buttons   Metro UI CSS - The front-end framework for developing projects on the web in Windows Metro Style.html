     Buttons

    With the Metro UI CSS you can easily create different types of buttons, stylized to Windows 8.

    Button styles can be applied to anything with the .button class applied. However, typically you'll want to apply these to only a and button elements for the best rendering.

    Push buttons

    Anchor
    Div
    Span 
    1. <button class="button">Caption</button>
    2. <button class="button" disabled>Caption</button>
    3. <div class="button">Caption</div>
    4. <a class="button">Caption</a>
    5. <span class="button">Caption</span>
    6. <button class="button rounded">Rounded</button>
    7. <button class="button cycle-button">+</button>
    8. <button class="button square-button">OK</button>

    With icons

    support both: icon fonts and images

    1. <button class="button"><span class="mif-checkmark"></span> Primary</button>
    2. <button class="button"><img src="images/Checkmark.png"> Primary</button>
