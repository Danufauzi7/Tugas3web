     RTL Support

    To use rtl support you must include metro-rtl.css in head of you page and set attribute dir to rtl for parent element or same element.

    Typography

    1. <link href="css/metro-rtl.css" rel="stylesheet">
    3. <div dir="rtl">
    4.     <h1>...</h1>
    5.     <p>...</p>
    6. </div>
    8. <p dir="rtl">...</p>
    10. <article dir="rtl">...</article>
